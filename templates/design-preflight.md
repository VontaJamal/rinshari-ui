# Design Preflight

Before UI/UX work in downstream repos, complete all of the following:

1. Read relevant docs in `design/rinshari-ui/principles/`.
2. Read local `docs/site-soul-brief.md`.
3. State intended visual/UX effect and how it supports user goals.
4. Identify one accessibility risk and mitigation.
5. In PR/task output, include:
   - Applied principles
   - Site Soul alignment
